# Calculator
This is a Calculator App

```bash
# Clone this project
$ git clone https://github.com/sethraj14/Calculator

# Access
$ cd Calculator

# Install dependencies
$ yarn install
$ cd ios && pod install

# Run the project in Xcode or Android Studio or
$ yarn ios // for iOS
$ yarn android // for Android

```
